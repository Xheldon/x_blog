  2. 장식 블록 & 목록

…너무 많아서 모두 보여줄 수 없습니다.

내 생각은?

중요한 부분:

  1. 장식, 포커스든 블록이든 blockquote로 Markdown 문법으로 변환되어야 합니다.

  2. 캡션은 해당하는 Markdown 문법이 없으며, 그냥 본문으로 렌더링되고 H5가 아닙니다.

  3. 코드 블록은 ```javascript과 같이 언어를 지정해야 합니다.

  4. 중첩 목록, 평면 목록이 아닙니다!

중요하지 않은 부분:

  1. 정렬은 무시합니다.

  2. 들여쓰기는 무시합니다.

  3. 수식은… 음 무시합니다.

내 계획은?

Craft 데이터를 Github Markdown으로 올바르게 변환하는 npm 패키지를 만들 예정입니다 🤪.
